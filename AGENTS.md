# AGENTS.md

Instructions for AI agents working in this repository.

## Git commits

- Write **single-line** commit messages — a concise subject only. No body, no bulleted description.
- **Never** add a `Co-Authored-By` trailer, a "Co-authored-by: <model>" line, or any "Generated with …" line to commits or PR descriptions.
- Keep the Conventional Commits style for the subject (e.g. `fix(persistence): reconcile adapter semantics`, `feat(core)!: curried defineStage`).
